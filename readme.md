用python写爬虫，爬取教务处网站的通知，对新通知进行提醒

## 方法

下载整个页面，选出位于<title>标签中的游泳信息，写入1.txt中，  
打开1.txt,读取第一行，与最新读取的网页中的第一个信息进行对比，   
如果不同则发送邮件，邮件内容为通知标题。
